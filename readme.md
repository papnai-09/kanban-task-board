Project Explanation

Project Name: Kanban Task Board
Tech Stack: React.js (Vite), useState, Props, LocalStorage

🔹 Overview

This project is a Trello-like task management board where users can organize tasks across three stages: To Do, In Progress, and Done.

🔹 Features

1. Add Task

Users can create new tasks using an input field
Tasks are added to the "To Do" column

2. Edit Task

Clicking on a task allows inline editing
Changes are saved instantly

3. Delete Task

Each task has a delete button to remove it

4. Move Task

Tasks can be moved between columns using buttons

5. Priority System

Tasks have priorities: High, Medium, Low
Color indicators:
High → Red
Medium → Orange
Low → Green

6. Search Functionality

Users can search tasks by name
Filtering is dynamic and case-insensitive

7. Local Storage

Tasks are saved in browser storage
Data persists after page refresh
🔹 UI/UX
Modern dark theme (inspired by Trello)
Responsive layout
Smooth hover animations
